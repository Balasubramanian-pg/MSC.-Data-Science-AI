# Page 1

Data Visualization 
& Storytelling
Presented by
ACV Subrahmanyam
Course Description
This course is designed to give participants a solid grounding in fundamental data
visualization and data storytelling learning concepts. 
The course covers :
Units
3 Credits
Exploratory vs. Explanatory analysis; 
Understanding the context for data presentations in 2D and 3D; 
Effective visuals; 
Gestalt principles of visual perception; 
Visual ordering, Decluttering, Storytelling; 
Visualization Design; 
Taxonomy of data visualization methods with the required libraries and tools; 
Bullet graphs, Pareto charts; 
Dashboard categorization, 
Characteristics of a well-designed dashboard and Key goals in the visual design
process; 
Techniques for visualization of high dimensional data & Discovered patterns; 
Effective Dashboard Display Media, Designing Dashboards for Usability; 
Common mistakes in Dashboards. 
Explore Tableau/ Python/ PowerBI to handle real-world datasets to visualize and
interpret.
