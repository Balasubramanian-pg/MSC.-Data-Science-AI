Visualisation or Dashboard Tools Overview Transcript 
 
We will now look at the different types of data visualisation tools that are out there in 
the market and which can help the analyst to build good data visualisation dashboards. 
These are two types of software that are available that help you build the 
visualisations. First is the low or no code utilities, then you have the utilities which are 
based on coding. 
  
Within the no or low code utilities, you have Tableau, Power BI and Google Data 
Studio. These are menu driven with pre-made visualisations for you to create 
dashboards and if you are convenient with basic data sheets like Google Sheets or 
Excel, you can create very interesting dashboards without much of coding abilities. All 
that you need to understand is how to use the preset menus and explore the 
functionalities to the fullest. 
  
You can build pre-customised dashboards and customise them for your specific 
purpose, make them live, connect with real data, refresh, share reports, so on and so 
forth. We will be seeing the application of all these three through our course and in our 
experiential learning platform so that you are well-versed with the utilisation of these 
software tools for building dashboards. As opposed to this, you have the code-based 
dashboard utilities, especially the Python and R-based utilities. 
  
In Python, you have Plotly and Voila Jupyter-based notebooks where you can create 
dashboards but that would require some amount of coding. In setting up the data 
structure, setting up the data frame, creating metrics, and using the attributes to create 
visualisations, however, you have the power of creating custom visuals and metrics 
using the code environment. And finally, you have R packages called ShinyR and 
FexDashboard which can be used for creating dashboards also.
