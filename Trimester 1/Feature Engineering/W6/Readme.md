

  
Week 6: Feature Engineering Techniques for Image Data
Module Description

This module focuses on extracting meaningful features from image data using several well-known techniques.

Learning Objectives

By the end of this module, you will be able to:

Understand the basics of image representation and the need for feature extraction in image data.
Learn techniques such as edge detection and corner detection.
Explore and implement Histogram of Oriented Gradients (HoG) for capturing shape and structure.
Demonstrate image feature engineering through hands-on examples and visualisations.
