

  
Week 3: General Feature Engineering Techniques
Module Description

This module introduces key techniques for creating, transforming and selecting features to improve model performance and interpretability.

Learning Objectives

By the end of this module, you will be able to:

Understand the concepts of feature extraction, construction and selection.
Learn and apply filter-based feature selection methods such as correlation metrics, chi-square tests and information-theoretic measures.
Explore wrapper-based methods like Sequential Forward and Backward Selection and understand their trade-offs.
Understand embedded methods and how they integrate feature selection into model training.
Evaluate and compare the effectiveness of different feature selection techniques in practical scenarios.
