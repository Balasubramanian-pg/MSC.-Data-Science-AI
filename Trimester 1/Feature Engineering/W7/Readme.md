

  
Week 7: Feature Engineering Techniques for Time-Series Data
Module Description

This module focuses on extracting and engineering features from time-series data to capture temporal patterns and improve predictive models.

Learning Objectives

By the end of this module, you will be able to:

Understand the challenges and methods for extracting features from time-series data.
Learn how to create time-based features such as lag features, rolling statistics, and window-based features.
Explore domain-specific feature creation techniques like heart rate variability, moving averages, and financial indicators (e.g., RSI, Bollinger Bands).
Apply temporal feature engineering techniques to real-world time-series datasets.
Evaluate the impact of engineered time-series features on model performance and forecasting accuracy.
