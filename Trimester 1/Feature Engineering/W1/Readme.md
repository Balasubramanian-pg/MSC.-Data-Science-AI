Week 1: Overview of Feature Engineering
Module Description

This module introduces the fundamental concepts and importance of feature engineering, and recognises the challenges of high-dimensional data.

Learning Objectives

By the end of this module, you will be able to:

Describe the role and purpose of feature engineering in the machine learning pipeline.
Interpret feature importance using simple, illustrative examples.
Apply basic feature engineering techniques to a small dataset.
Identify and explain common issues arising from high-dimensional data.
Summarise general strategies for effective feature engineering.
