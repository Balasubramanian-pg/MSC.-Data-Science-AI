

  
Week 8: Automated Feature Engineering
Module Description

This module introduces automated feature engineering tools to streamline the feature generation process and enhance model development.

Learning Objectives

By the end of this module, you will be able to:

Understand the benefits and limitations of automated feature engineering tools.
Explore and apply Featuretools for automatic feature extraction and engineering.
Learn how to use TSFresh to extract relevant features from time-series data.
Discover Featurewiz for feature selection and preprocessing in machine learning workflows.
