

  
Week 2: Handling Numeric Data
Module Description

This module covers essential techniques for preparing numeric data for analysis and machine learning. You’ll learn how to discretise continuous features, apply feature scaling, use smoothing methods, perform attribute transformations, and understand the impact of scaling on proximity-based models.

Learning Objectives

By the end of this module, you will be able to:

Understand common numerical data transformation techniques such as discretisation, normalisation, and smoothing.
Apply feature scaling methods like min-max scaling, standardization and robust scaling.
Explore attribute transformation methods including logarithmic, polynomial and power transformations.
Analyse how preprocessing choices affect distance-based models and similarity metrics.
Evaluate the impact of numerical feature engineering on model performance.
